## 这是什么

Miampic 是一款 Chrome 浏览器插件。

看到一张喜欢的图，想用 AI 生成同款？以前你需要把图片拷贝出来、发给 AI 分析提取图片元素风格等、再转发给生图工具，链路又长又繁琐。

Miampic 把这个过程压缩成一步——截图、上传或右键图片，直接拿到可用的提示词。

---

## 功能

**三种图片输入方式**
- 📷 截图当前页面
- 🖼 上传本地图片
- 🖱 右键网页图片直接生成

**两种提示词用途**
- **直接生图**：提取参考图风格，发给 AI 直接生成类似图片
- **辅助改图**：搭配自己的照片，复刻属于自己的同款 AI 美图
- 可使用豆包 / 即梦 / 海螺 / 通义万象 / Midjourney / Stable Diffusion 等生图工具

**其他**
- 支持选择生图提示词的语言（中文 / 英文）
- 支持翻译生成后的提示词
- 历史记录本地保存，支持手动清理记录

## 安装方法

目前尚未上架 Chrome 应用商店，请使用开发者模式手动安装：

1. 下载本仓库（点击右上角 **Code → Download ZIP**，解压）
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角开启**开发者模式**
4. 点击**加载已解压的扩展程序**
5. 选择解压后的 `miampic-extension` 文件夹
6. 安装完成，浏览器工具栏出现 Miampic 图标

---

## 使用方法

**首次使用**
1. 点击工具栏的 Miampic 图标
2. 点击右上角设置 ⚙️，填入你的 API Key
3. 返回主界面即可使用

**生成提示词**
1. 选择提示词用途：「直接生图」或「辅助改图」
2. 选择图片来源：截图 / 上传 / 右键图片
3. 等待 AI 分析，复制生成的提示词
4. 粘贴到你喜欢的生图工具，开始创作

**查看历史**
- 点击设置页，可查看所有历史提示词，支持手动清理
